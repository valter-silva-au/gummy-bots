# Gummy Bots 🫧

> *Flick tasks at your AI bot like a billiards game.*

A gamified AI assistant where tasks orbit your bot as colorful bubbles. Flick them to execute. Physics-based, haptic-rich, dopamine-approved.

## What is this?

**Gummy Bots** turns boring task management into a tactile game. Your AI bot sits at the center of the screen. Tasks appear as colorful "gummies" orbiting around it. Flick a gummy at the bot — it catches it, executes the task, done.

The interaction model is inspired by **bilhar/gude** — Brazilian street marble games where you flick with precision at a target.

## Key Features

- 🤖 **AI Bot** — LLM-powered central orb that executes your tasks
- 🫧 **Gummies** — Color-coded task bubbles (blue=comms, green=calendar, orange=info, red=urgent, purple=automation)
- 🎯 **The Flick** — Physics-based gesture to delegate tasks to your bot
- 🔌 **Connectors** — Gmail, Calendar, Slack, Notion feed tasks automatically
- 🎮 **Gamification** — XP, levels, streaks, bot evolution, achievements
- 📳 **Haptics** — Satisfying tactile feedback on every interaction

## Tech Stack

- **Frontend:** React Native / Expo
- **Animations:** React Native Reanimated + Skia
- **Gestures:** React Native Gesture Handler
- **Backend:** LLM-powered agentic engine
- **Integrations:** OAuth2 connectors
