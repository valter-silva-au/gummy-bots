# Project Brief — Gummy Bots

## Vision
A mobile app that gamifies task *execution* (not tracking) through physics-based interactions with an AI bot.

## Core Requirements
1. Central AI bot orb with personality states (idle → thinking → working → celebrating)
2. Color-coded task gummies orbiting the bot with physics
3. Flick gesture to delegate tasks — the physical gesture IS the authorization
4. OAuth2 connectors to feed gummies from real services (Gmail, Calendar, Slack)
5. Gamification layer: XP, levels, streaks, bot evolution, achievements
6. Multisensory feedback: visual pop + ASMR audio + haptic vibration
7. Edge AI for privacy-first task triage

## Target Users
- Action-oriented productivity users
- ADHD/executive dysfunction (spatial UI, no text walls)
- Automation enthusiasts
- Mobile-first (iOS + Android)

## Business Model
- Free: 3 connectors, core mechanics
- Pro: $9.99/mo unlimited connectors + advanced automations
- Bot Skins: $1.99-$4.99 cosmetic IAP
- Sound Packs: $0.99-$2.99 custom ASMR pops
- Team Mode (v3+): $14.99/seat/mo

## Non-Goals (v1)
- No WhatsApp integration (Meta policy ban)
- No Team Mode (requires SOC2/GDPR)
- No web version (mobile-first)
